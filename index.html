Sınav Sistemi API

Bu sunucu, Sınav Sistemi uygulamasının backend hizmetini sağlamaktadır.

API Endpoint'leri

GET /api/exams

Tüm sınavların listesini getirir.

GET /api/exams/:id

Belirli bir sınavın detaylarını getirir.

POST /api/auth/login

Kullanıcı girişi yapar ve token döndürür.

POST /api/exams/:id/start

Sınavı başlatır.

POST /api/exams/results/:examResultId/submit

Sınav cevaplarını gönderir.
